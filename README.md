# 🛠 프로젝트 설정 개요

## ✅ 셋팅되어있는 것

- **빌드 도구:** Vite

- **패키지 매니저:** pnpm

- **스타일:** Tailwind CSS

- **린팅:** ESLint

- **포매터:** Prettier

- **라우팅:** react-router-dom
